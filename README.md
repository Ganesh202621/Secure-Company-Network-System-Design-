# Secure Company Network System Design Using Cisco Packet Tracer

## 📌 Project Overview

This project demonstrates the design and implementation of a secure enterprise network infrastructure using Cisco Packet Tracer. The network architecture includes routing, switching, firewall security, wireless networking, VoIP services, and cloud integration following hierarchical design principles.

---

## 🎯 Objectives

- Design a secure enterprise network
- Implement routing and switching technologies
- Configure firewall security and access policies
- Enable high availability and redundancy
- Integrate wireless, VoIP, and cloud services
- Improve scalability and reliability

---

## 🛠 Technologies & Tools Implemented

- Cisco Packet Tracer
- Cisco ASA Firewall
- OSPF Routing Protocol
- Default Static Routing
- NAT
- ACL (Access Control Lists)
- VLANs
- Router-on-a-Stick Inter-VLAN Routing
- SVI
- HSRP
- EtherChannel (LACP)
- STP PortFast
- BPDU Guard
- Cisco Wireless LAN Controller (WLC)
- Lightweight Access Points
- Active Directory DHCP Server
- Cisco Voice Gateway
- IP Phones
- SSH Remote Access
- AWS Cloud Platform
- ISP Connectivity

---

## 🔍 Features Implemented

- Secure Internal, External, and DMZ network zones
- High availability using HSRP
- Link redundancy with EtherChannel
- Multiple network segments:
  - Corporate Network
  - Employee Network
  - Guest WiFi
  - External Auditor Network
- VoIP communication
- Wireless connectivity
- Secure remote administration
- Cloud integration
